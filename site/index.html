ice libs








What is ice?

Collection of Single-Header Cross-Platform ANSI C (C89) libraries, That are:




Portable

Powered by ANSI C (C89), The first (and oldest) official C standard which works with almost any C compiler!

C++/Objective-C Compatible

Can be used with both C++ and Objective-C




Cross-Platform

Build efficient softwares powered by user-friendly APIs, that runs on Desktops, Mobiles, Web, Game Consoles, and even Embedded Platforms!




Memory-Efficient

Memory-Safe with no leaks, Allows usage of custom memory allocators!




Configurable

Allows to customize calling convention and platform implementations and compilation flags!




User-Friendly

Their APIs are easy to learn and well-documented, Making C programming more fun!




Free

Freeware, You don't have to pay for anything!




Bindings-Friendly

Enjoy using them with your favourite language, Bindings for LOT of programming languages!
